The 'java' subdirectory contains the Java source code
for the Midiki libraries and the sample dialogue managers.

The 'gc' subdirectory contains files used for interfacing
with Galaxy Communicator.

The 'oaa' subdirectory contains files used for interfacing
with the Open Agent Architecture.

The 'org' subdirectory contains legacy code associated
with scripting languages and code generators for Midiki.
That legacy code is currently unmaintained, but has been
retained within the repository for possible future work.
It may be modified or deleted in future releases without
prior notice.
